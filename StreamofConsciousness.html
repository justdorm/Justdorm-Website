Stream of Consciousness is an interactive installation that maximizes user engagement and customization with AI image generation. The system consists of a webcam where users can physically interact with the AI process, and a bluetooth controller where they can change the installation settings to suit their imagination. In Stream of Consciousness, the user becomes both the artist and the art.

Stream of Consciousness was developed in TouchDesigner using StreamDiffusion, a real-time implementation of Stable Diffusion. The image generation model that this iteration is running is SDTurbo. For the user silhouette isolation, I am using Nvidia Background Removal. The silhouette is composited with a foreground and background noise and is then used as an image to image input. The system also includes both a preset management system for an intuitive and curated experience, as well as in depth UI to fully customize the prompt and generation settings for users who want full creative control.
