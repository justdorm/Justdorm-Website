Stream of Consciousness

A real-time AI installation where the user becomes both the artist and the art

Role
Artist & Creative technologist
Tools
TouchDesigner · StreamDiffusion · SDTurbo
Venue
SVA MFA Computer Arts

Infinite variation, instantly rendered

Stream of Consciousness is an interactive installation that lets people shape AI-generated imagery in real time. A webcam puts the user into the image, and a Bluetooth controller lets them change the installation's settings on the fly.

In Stream of Consciousness, the user becomes both the artist and the art.

How it works

The installation was developed in TouchDesigner using StreamDiffusion, a real-time implementation of Stable Diffusion, with SDTurbo as the generation model. Nvidia Background Removal isolates the user's silhouette, which is composited with foreground and background noise and used as the image-to-image input.

A preset system makes it easy to jump in, while a full UI exposes the prompt and generation settings for anyone who wants complete control.
