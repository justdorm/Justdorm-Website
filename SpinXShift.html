Spin X Shift

360° audio-reactive visuals with a live depth camera, wrapped around a room with eleven projectors

Role
Visual artist & creative technologist
Tools
TouchDesigner · Depth camera · MIDI · NDI · Pixera
Venue
Porsche Studio Portland — Shifting Realities Art Show

Immersive Real-Time Environments

I had the opportunity to create visuals for a DJ event at Porsche Studio in Portland, Oregon as part of their Shifting Realities Art Show. Working from New York, I prepared three sets of visuals for the DJs in TouchDesigner: a neon-light-inspired set to start the night, an Egypt-meets-Portland landscape scene for Motaz, and an abstract carpentry-inspired set for Chi Duly.

Each set reacts to the audio, creating a real-time rendered immersive environment. A depth camera feeds a live 3D representation of the DJ into the scene, distorting in time with the music — and a MIDI controller is bound to different aspects of each scene, allowing for variation and art direction throughout each performance.

How it works

Using the dimensions of the space, I built a pre-visualization in New York that previewed exactly how the projections would look on Porsche Studio's walls before I ever stepped into the room.

On site, the first task was bridging my TouchDesigner system into the Pixera media server that maps the space's eleven projectors — connected over NDI. Motaz's DJ deck ran into the house mixer, with a copy of the audio routed to the server running TouchDesigner. The depth camera was processed on a laptop near the booth, its signal sent over Ethernet to the server.
